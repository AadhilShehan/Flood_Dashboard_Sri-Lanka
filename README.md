# Flood_Dashboard_Sri-Lanka
## Overview
This project presents an interactive dashboard developed using Streamlit to analyse flood events in Sri Lanka. The dashboard provides insights into flood-affected areas, population exposure, and agricultural impact. The aim is to support decision-makers in understanding flood patterns and planning effective responses.
## Dataset
Source : Humanitarian Data Exchange (HDX)
Description: Biweekly flood event data for Sri Lanka including:
  - Flooded area
  - Cropland affected
  - Population exposed
  - District-level data over time
## Objectives
- Analyse flood trends over time  
- Identify high-risk districts  
- Evaluate population exposure  
- Assess agricultural damage
## Features of the dashboard
- Interactive filters (district and date range)  
- Key performance metrics  
- Flood trend analysis (line chart)  
- Population exposure distribution (pie chart)  
- Agricultural impact heatmap
## Conclusion
The dashboard provides meaningful insights into flood impacts in Sri Lanka. It highlights vulnerable regions and supports better decision-making for disaster management and sustainability planning.
